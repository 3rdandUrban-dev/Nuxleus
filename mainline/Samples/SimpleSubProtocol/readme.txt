How to use this SubProtocol?

1. Copy this project's output to working directory of SuperWebSocket's running container,
SuperWebSocketService, SuperWebSocketWeb or your own container.

2. Add attrubute subProtocol="SuperWebSocket.Samples.SimpleSubProtocol.SimpleBasicSubProtocol, SuperWebSocket.Samples.SimpleSubProtocol" to the websocket server instance's configuration element of SuperWebSocket's running container,
like the file SuperWebSocket.Service.exe.config in current project.
