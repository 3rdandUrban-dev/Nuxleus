Mvp.Xml Project

This project is developed by Microsoft MVPs in XML technologies and XML Web Services worldwide. It is aimed at supplementing .NET framework functionality available through the System.Xml namespace and related namespaces such as System.Web.Services. It is documented extensively through weblog posts. All the project's classes contain extensive tests to ensure its quality, as well as the peer review among this highly focused group of XML lovers.
